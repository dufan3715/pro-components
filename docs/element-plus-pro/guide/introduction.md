# 介绍

## 什么是 element-plus-pro？

`@qin-ui/element-plus-pro` 是基于 Element Plus 的 Pro 组件封装，提供 `ProForm`、`ProTable`、`ProComponentProvider` 与对应 hooks。

## 特性

- 基于 Element Plus 2.13.6+
- 提供 ProForm / ProTable / ProComponentProvider
- 支持配置化表单字段与表格列
- 支持 `columns` 内通过 `formatter` 与 `render(scope)` 自定义渲染
- 支持 `componentVars` 全局注入默认行为

## 支持环境

- Vue 3.5+
- Element Plus 2.13.6+
- 现代浏览器（ES2019+）
