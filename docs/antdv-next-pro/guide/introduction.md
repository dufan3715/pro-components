# 介绍

## 什么是 antdv-next-pro？

`antdv-next-pro` 是一个基于 antdv-next 的高级组件库，提供了更强大的表单和表格功能。它通过封装常用的业务场景组件，帮助开发者快速构建企业级中后台产品。

## 特性

- 🛡️ 基于 antdv-next 封装
- 🔥 提供 ProForm 和 ProTable 高级组件
- ⚙️ 支持表单字段动态生成、表格列动态配置
- 📦 轻量、易用
- 🌍 支持 **ES Module**

## 支持环境

- Vue 3.5+
- antdv-next 1.x
- 现代浏览器（支持 ES6+）

## 组件列表

| 组件名               | 说明               |
| -------------------- | ------------------ |
| ProComponentProvider | 全局组件配置提供者 |
| ProForm              | 高级表单组件       |
| ProTable             | 高级表格组件       |

## 设计理念

`antdv-next-pro` 的设计遵循以下原则：

1. **配置化**：通过配置对象而非模板代码来定义组件行为
2. **约定优于配置**：提供合理的默认值，减少不必要的配置
3. **易于集成**：与 antdv-next 无缝集成，保持一致的 API
4. **可扩展**：支持自定义组件和样式，满足个性化需求
